02 — инструменты

под капотом

Мощные инструменты EliteClean для продвинутой работы с системой

Чистка USN журнала
Удаление записей из журнала NTFS USN. Скрывает факты создания, изменения или удаления файлов на диске.
Очистка следов
Комплексная очистка:
  • →ShellBag — история папок в проводнике
  • →Prefetch — данные о запущенных программах
  • →Minidump — файлы отладки при сбоях
Глобальная очистка
Полная очистка с выбором компонентов:
🗑️ корзина
📝 temp
🌐 кэш браузеров
📋 clipboard
📁 recent
🔍 history
📊 error reports
🔧 логи
Очистка памяти javaw
Сканирование процессов javaw.exe и освобождение занятой памяти. Полезно для Minecraft и других Java-приложений.
Симуляция папок
Создание фиктивной структуры папок и файлов для имитации активности пользователя. Маскировка реальных действий.
